# React CRUD App

A simple, extensible User Management application built with React, Tailwind CSS, and JSON Server.

## Features
- **CRUD Operations**: Create, Read, Update, Delete users.
- **Extensible Forms**: Configuration-driven forms (`DynamicForm`) that allow adding new fields easily.
- **Validation**: Regex-based validation for emails, phone numbers, and names.
- **Modern UI**: Clean design with Tailwind CSS v4, smooth animations (`tailwindcss-animated`), and Toast notifications (`react-toastify`).

## Quick Start

### 1. Backend
```bash
cd backend
npm install
npm run dev
```
Runs on `http://localhost:5000`

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs on `http://localhost:5173` (typically)
